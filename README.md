# Adaptive Sandbox Kit

## Core Principles
1. **Partitioned memory** — What happens in sandbox stays in sandbox
2. **Mentor model** — Guided exploration, not empty canvas
3. **No fear** — Org data can be used for simulation without risk
4. **Configurable** — Small orgs can self-serve; enterprise can customize

## Target Users
- Small/mid-size orgs who need confidence-building sandboxes but can't afford enterprise implementation

## Open Questions
- What's the minimal viable version? (1 hour setup? 1 day?)
- How does mentor model work without live humans?
- Can we template common workflows (gov procurement, NGO reporting, etc.)?
